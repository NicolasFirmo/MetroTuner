## About

A simple audio tuner app for Android.

## Build

### Requirements

- Android Studio (tested on version 2021.2.1.14).
- CMake.

### Instructions

1. Open the project on Android Studio.
2. Upgrade gradle when asked.
3. Make Project (Ctrl+F9).
